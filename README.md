# Proyecto integrador

Para las materias de programacion III, base de datos II.


## Iniciar el servidor local

Previamente necesitamos tener instalado node.js. Para empezar, just clone the repository and run `npm install && npm run dev`:

    git clone https://github.com/gonchi028/restaurante-proyecto-integrador.git
    npm install
    npm run dev

## Configuracion previa

Antes de poder hacer el `npm run dev` debemos crear el archivo env.local con las siguientes llaves:

    NEXT_PUBLIC_SUPABASE_URL=llave_1
    NEXT_PUBLIC_SUPABASE_ANON_KEY=llave_2

que se encuentran en teams.
